## 문제해결을 위한 빅데이터 활용 프로젝트

#### 1.Project

- 서울시 지역구별 치안 요인 분석

#### 2. Description

- 지역 치안에 영향을 주는 요인을 바탕으로 서울시 지역구별 치안 수준 분석을 위한 학습데이터를 구축하고 시각화

- 서울시 치안 관련 데이터 크롤링
- 데이터 전처리
- 수집 정보 시각화
- Database구축
- AWS 개발 환경 구축
- 결과 웹 페이지 구축

#### 4. Skill

- AWS
